# DSDigitalz Portfolio

A modern personal portfolio website built with **Next.js 16**, **TypeScript**, and **Tailwind CSS 4**. This project includes animated UI sections, a responsive landing page, reusable components, and a polished dark theme.

## 🚀 Overview

This single-page portfolio features:

- Hero section with animated profile and CTA buttons
- About section with experience, education, and location cards
- Skills section displaying frontend, backend, design, and tools
- Projects gallery with live links to portfolio work
- Contact section with direct email, phone, location, and a contact form UI

## ✨ Highlights

- Responsive layout for desktop and mobile
- Framer Motion animations throughout the page
- Tailwind CSS 4 styling with custom CSS variables
- Google fonts loaded with Next.js font optimization
- Mobile menu and sticky animated header
- Built as a static frontend portfolio using Next.js app router

## ⚙️ Run Locally

```bash
git clone https://github.com/DsDigitalz/Apx-PortfolioWebsite.git
cd apx-portfolio-website
npm install
npm run dev
```

Then open **http://localhost:3000**.

## 📦 Available Scripts

- `npm run dev` — start the local development server
- `npm run build` — build the production app
- `npm run start` — start the production server
- `npm run lint` — run ESLint checks

## 📌 Notes

- The app uses the App Router and absolute imports.
- The contact form is currently UI-only and does not submit to a backend.
- Project cards link to live demos and design previews.
